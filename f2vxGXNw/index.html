470. 用Rand7()实现Rand10()#

链接: https://leetcode.cn/problems/implement-rand10-using-rand7/description/

1、题目描述#

给定方法 rand7 可生成 [1,7] 范围内的均匀随机整数，试写一个方法 rand10 生成 [1,10] 范围内的均匀随机整数。

你只能调用 rand7() 且不能调用其他方法。请不要使用系统的 Math.random() 方法。

每个测试用例将有一个内部参数 n，即你实现的函数 rand10() 在测试时将被调用的次数。请注意，这不是传递给 rand10() 的参数。

示例 1:

输入: 1

输出: [2]

示例 2:

输入: 2

输出: [2,8]

示例 3:

输入: 3

输出: [3,8,10]

提示:

1 <= n <= 105

进阶:

  • rand7()调用次数的 期望值 是多少 ?
  • 你能否尽量少调用 rand7() ?

2、思路说明#

该题目使用拒绝采样方法。

使用公式 rand7()*rand7() 得到的结果可能有49种，如下面的矩阵所示。使用 (row-1)*7+col 得到的就是下述矩阵中每个元素的索引。拒绝采样就是仅保留自己想要的值，丢弃自己不想要的值。

在下述矩阵中，仅保留索引为1到10的位置的元素，其他的所有的元素都丢弃，最终结果是没有问题的，示例代码如下：

def rand10(self):
    while True:
        row = rand7()
        col = rand7()
        idx = (row-1)*7+col
        if idx <= 10:
            return idx

但是可以看到下述矩阵中有49个元素，只使用前10个元素，其他的元素都丢弃的效率有点低，所以可以改进为使用前40个元素。详细代码见下一小节。

3、代码#

# The rand7() API is already defined for you.
# def rand7():
# @return a random integer in the range 1 to 7

class Solution:

    def rand10(self):
        """
        :rtype: int
        """

        while True:
            row = rand7()
            col = rand7()
            idx = (row-1)*7+col
            if idx <= 40:
                return int((idx-1)/4)+1
