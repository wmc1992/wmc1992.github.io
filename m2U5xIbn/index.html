72. 编辑距离#

链接: https://leetcode.cn/problems/edit-distance/description/

1、题目描述#

给你两个单词 word1 和 word2， 请返回将 word1 转换成 word2 所使用的最少操作数 。

你可以对一个单词进行如下三种操作：

  • 插入一个字符
  • 删除一个字符
  • 替换一个字符

示例 1：

输入：word1 = "horse", word2 = "ros"

输出：3

解释：

horse -> rorse (将 'h' 替换为 'r')

rorse -> rose (删除 'r')

rose -> ros (删除 'e')

示例 2：

输入：word1 = "intention", word2 = "execution"

输出：5

解释：

intention -> inention (删除 't')

inention -> enention (将 'i' 替换为 'e')

enention -> exention (将 'n' 替换为 'x')

exention -> exection (将 'n' 替换为 'c')

exection -> execution (插入 'u')

提示：

  • 0 <= word1.length, word2.length <= 500
  • word1 和 word2 由小写英文字母组成

2、思路说明#

该题目使用动态规划求解。

dp数组的含义：dp[i][j] 表示字符串 word1[:i] 与字符串 word2[:j] 的编辑距离。

为了方便处理，dp数组的大小为：(len(word1)+1, len(word2) + 1)，多出来的元素用于存放字符串为空字符串时的边界条件。

字符串 word1[:i] 与字符串 word2[:j] 的编辑距离 可以通过如下几种方式计算得到：

  • 字符串 word1[:i-1] 与字符串 word2[:j] 的编辑距离 加上1。这里的编辑距离增加1对应的操作是：给字符串 word1[:i-1] 增加一个字符；

  • 字符串 word1[:i] 与字符串 word2[:j-1] 的编辑距离 加上1。这里的编辑距离增加1对应的操作是：给字符串 word2[:j-1] 增加一个字符；

  • 字符串 word1[:i-1] 与字符串 word2[:j-1] 的编辑距离 加上1或者0。这里的编辑距离增加1或者0对应的操作是：

    • 如果字符 word1[i] 与字符 word2[j] 不同，那么已知了 字符串 word1[:i-1] 与字符串 word2[:j-1] 的编辑距离 之后，字符串 word1[:i] 与字符串 word2[:j] 的编辑距离 只需要修改一下 word1[i] 与 word2[j] 为同一个字符就可以了，所以是编辑距离增加1；

    • 如果字符 word1[i] 与字符 word2[j] 相同，那么 字符串 word1[:i] 与字符串 word2[:j] 的编辑距离 与 字符串 word1[:i-1] 与字符串 word2[:j-1] 的编辑距离 就是相同的，所以是编辑距离加0；

所以递推公式如下：

dp[i][j] = \begin{cases} \text{min}(dp[i-1][j] + 1, dp[i][j-1] + 1, dp[i-1][j-1] + 1) &\text{if } \text{word1}[i] \neq \text{word2}[j]\\ \text{min}(dp[i-1][j] + 1, dp[i][j-1] + 1, dp[i-1][j-1]) &\text{if } \text{word1}[i] = \text{word2}[j]\\ \end{cases}

边界条件是当 i 等于 0，或者 j 等于 0 时，此时一个字符串为空字符串，两个字符串之间的编辑距离就是另一个非空字符串的长度。

3、代码#

class Solution:

    def mymin(self, a, b, c):
        x = a if a < b else b
        x = x if x < c else c
        return x

    def minDistance(self, word1: str, word2: str) -> int:
        dp = [[0 for _ in range(len(word2) + 1)] for _ in range(len(word1) + 1)]

        # 边界条件
        for i in range(len(word1) + 1):
            dp[i][0] = i
        for j in range(len(word2) + 1):
            dp[0][j] = j

        # 递推公式
        for i in range(len(word1) + 1):
            for j in range(len(word2) + 1):
                if i == 0 or j == 0:
                    continue

                if word1[i-1] == word2[j-1]:
                    dp[i][j] = self.mymin(dp[i-1][j] + 1, dp[i][j-1] + 1, dp[i-1][j-1])
                else:
                    dp[i][j] = self.mymin(dp[i-1][j] + 1, dp[i][j-1] + 1, dp[i-1][j-1] + 1)
        return dp[len(word1)][len(word2)]
