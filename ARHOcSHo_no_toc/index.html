144. 二叉树的前序遍历#

链接: https://leetcode.cn/problems/binary-tree-preorder-traversal/

1、题目描述#

给你二叉树的根节点 root ，返回它节点值的 前序 遍历。

示例 1：

输入：root = [1,null,2,3]

输出：[1,2,3]

示例 2：

输入：root = []

输出：[]

示例 3：

输入：root = [1]

输出：[1]

示例 4：

输入：root = [1,2]

输出：[1,2]

示例 5：

输入：root = [1,null,2]

输出：[1,2]

提示：

  • 树中节点数目在范围 [0, 100] 内
  • -100 <= Node.val <= 100

2、思路说明#

递归方法比较简单，略过。

非递归方法使用的数据结构是栈，由于前序遍历的顺序是根左右，这个直接符合获取树中各个节点的顺序，所以每次就是先处理当前节点，然后分别将左孩子和右孩子入栈就可以了。

3、代码#

from typing import Optional, List


# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:

    # 递归方法
    def recursive(self, node, result):
        if node is None:
            return

        result.append(node.val)
        self.recursive(node.left, result)
        self.recursive(node.right, result)

    def preorderTraversal2(self, root: Optional[TreeNode]) -> List[int]:
        result = list()
        self.recursive(root, result)
        return result

    # 非递归方法
    def preorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        # 使用栈实现前序遍历，入栈时先入栈右子树，后入栈左子树，可以保证出栈时先出栈左子树，后出栈右子树
        result = []
        if root is None:
            return result

        # 使用列表实现栈，入栈 stack.append()，出栈 stack.pop()，查看栈顶元素 stack[-1]
        stack = list()
        stack.append(root)

        while len(stack) > 0:
            node = stack.pop()
            result.append(node.val)

            if node.right is not None:
                stack.append(node.right)
            if node.left is not None:
                stack.append(node.left)
        return result
