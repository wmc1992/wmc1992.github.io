49. 字母异位词分组#

链接: https://leetcode.cn/problems/group-anagrams/description/

1、题目描述#

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单词。

示例 1:

输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"] 输出: [["bat"],["nat","tan"],["ate","eat","tea"]]

示例 2:

输入: strs = [""] 输出: [[""]]

示例 3:

输入: strs = ["a"] 输出: [["a"]]

提示：

1 <= strs.length <= 104 0 <= strs[i].length <= 100 strs[i] 仅包含小写字母

2、思路说明#

这个题目就是单纯的考一下数据结构，使用一个字典解决该问题。字母异位词 是所有的字母都相同，所以对字符串的每个字符进行排序作为 key，就可以直接把字母异位词分组了。

3、代码#

from typing import List

class Solution:

    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        mydict = dict()

        for s in strs:
            k = "".join(sorted(s))
            if k not in mydict:
                mydict[k] = []
            mydict[k].append(s)

        return list(mydict.values())
