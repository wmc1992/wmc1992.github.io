94. 二叉树的中序遍历#

链接: https://leetcode.cn/problems/binary-tree-inorder-traversal/

1、题目描述#

给定一个二叉树的根节点 root ，返回 它的 中序 遍历 。

示例 1：

输入：root = [1,null,2,3]

输出：[1,3,2]

示例 2：

输入：root = []

输出：[]

示例 3：

输入：root = [1]

输出：[1]

提示：

  • 树中节点数目在范围 [0, 100] 内
  • -100 <= Node.val <= 100

2、思路说明#

递归遍历的思路比较简单，没什么好说的。

非递归遍历时使用的数据结构是栈（前中后序遍历使用的数据结构都是栈，只是遍历时的处理方式不同），中序遍历的顺序是左中右，所以要先处理所有的左孩子，再处理根节点。提现在代码中就是：在每次大循环中，先使用一个小循环把所有的左孩子都入栈，然后再处理当前节点。

注意：另外，在这个代码中，node 这个变量的使用方式和之前的常规思路不太一致，所以这个代码最好直接背下来。

3、代码#

from typing import Optional, List

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:

    # 递归遍历
    def recursive(self, node, result):
        if node is None:
            return
        self.recursive(node.left, result)
        result.append(node.val)
        self.recursive(node.right, result)

    def inorderTraversal2(self, root: Optional[TreeNode]) -> List[int]:
        result = []
        self.recursive(root, result)
        return result

    # 非递归遍历
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        """
        注意中序遍历是左中右，也就是说要求每个当前节点必须在左孩子处理完之后才能够进行处理。

        体现在代码中就是，在每个外层循环中要先将所有的左孩子都入栈之后，再处理当前节点。
        """
        result = []

        node = root
        stack = []
        while len(stack) > 0 or node is not None:
            while node is not None:
                stack.append(node)
                node = node.left

            node = stack.pop()
            result.append(node.val)

            node = node.right
        return result
