121. 买卖股票的最佳时机#

链接: https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/

1、题目描述#

给定一个数组 prices ，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。

你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0 。

示例 1：

输入：[7,1,5,3,6,4]

输出：5

解释：在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。

注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

示例 2：

输入：prices = [7,6,4,3,1]

输出：0

解释：在这种情况下, 没有交易完成, 所以最大利润为 0。

提示：

  • 1 <= prices.length <= 105
  • 0 <= prices[i] <= 104

2、思路说明#

该题目是经典的动态规划题目。主要是分析dp数组如何设计。在该题目中，买入的时间一定是在卖出时间前的。无论是在哪一天卖出，想要获得在该天卖出的最大利润，就需要在该天之前的价格最低的那一天买入。

根据上述分析，就把dp数组设计为：dp[i]表示假设在第i这一天卖出，那么在第i这一天的前面中哪一天的价格最低，dp[i]中存储的是价格最低的这一天是哪天。

3、代码#

from typing import List


class Solution:

    def maxProfit(self, prices: List[int]) -> int:
        # 该dp数组中的值为-1时表示最终的利润为0
        dp = [-1 for _ in range(len(prices))]

        for i in range(len(prices)):

            # 边界条件，在第 0 天卖出是不被允许的
            if i == 0:
                dp[i] = -1
                continue

            if dp[i-1] == -1:
                if prices[i-1] < prices[i]:
                    dp[i] = i - 1
                else:
                    dp[i] = -1
            else:
                if prices[dp[i-1]] < prices[i-1]:
                    dp[i] = dp[i-1]
                else:
                    dp[i] = i-1

        max_profit = 0
        for i in range(len(dp)):
            if dp[i] == -1:
                continue
            profit = prices[i] - prices[dp[i]]
            if profit > max_profit:
                max_profit = profit

        return max_profit
