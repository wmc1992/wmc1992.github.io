51. N 皇后#

链接: https://leetcode.cn/problems/n-queens/

1、题目描述#

按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

n 皇后问题 研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 n ，返回所有不同的 n 皇后问题 的解决方案。

每一种解法包含一个不同的 n 皇后问题 的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位。

示例 1：

输入：n = 4

输出：[[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]

解释：如上图所示，4 皇后问题存在两个不同的解法。

示例 2：

输入：n = 1

输出：[["Q"]]

提示：

  • 1 <= n <= 9

2、思路说明#

能否仅用for循环解决#

在没有系统学习该类问题前，总想使用for循环解决该问题，那么能否仅使用for循环解决该问题？肯定是不能。

如果仅使用for循环，那么一般会怎样做呢？首先，外层for循环遍历每一行。然后就没有办法处理了。对于第一行，总共有 N 个格子，每个格子都有可能放置为皇后，那么第一行需要一个for循环遍历。如果对第一行使用了一个for循环做遍历，那么第二行也有 N 个格子，也是每个格子都有可能放置为皇后，那么第二行也需要一个for循环遍历。这样看起来需要 N 的for循环。这种代码是写不出来的。

下面放一张代码随想录中的图片直观感受一下这个过程：

为何该问题无法仅使用for循环解决。因为这个问题的时间复杂度为阶乘复杂度 O(n!)，仅使用for循环的话，无论有再多个有限的for循环都是多项式复杂度 O(n^k)，其中 k 为常数。

这里再强调一下多项式时间复杂度、指数时间复杂度、阶乘时间复杂度之间的关系为：O(n^k) < O(k^n) < O(n!)，其中 k 都是常数。

伪代码#

接下来看一个阶乘复杂度的问题如何遍历其所有的可能的状态。注意N皇后问题只要求解出来结果就行，不要求对其做优化，其已经是 O(n!) 复杂度的问题了。

  • 初始化一个二维矩阵表示棋盘

  • 回溯入口：

    • 记当前行为 row

    • 边界条件：如果 row 已经大于总行数，那么就无需继续回溯了；

    • 遍历当前行的每一个格子，每个格子的纵坐标为 col：

      • 判断当前格子是否能够放置皇后：若不能则直接continue；若能则继续执行下面的过程；

      • 将当前格子放置皇后

      • 到回溯入口处开始回溯，此次回溯的行数为 row+1

      • 待所有后续状态都回溯完成之后，将当前格子恢复为不放置皇后

3、代码#

class Solution:
    result = list()

    def is_valid(self, grid, row, col):
        """
        需要判断的是正上方、左上方、右上方三个方向

        正下方、左下方、右下方、以及同一行是不需要判断的
        """

        # 正上方
        for i in range(row):
            if grid[i][col] == "Q":
                return False
        # 左上方
        i, j = row - 1, col - 1
        while i >= 0 and j >= 0:
            if grid[i][j] == "Q":
                return False
            i -= 1
            j -= 1
        # 右上方
        i, j = row - 1, col + 1
        while i >= 0 and j < len(grid[0]):
            if grid[i][j] == "Q":
                return False
            i -= 1
            j += 1

        return True

    def bachtrace(self, grid, row):
        if row >= len(grid):
            self.result.append(["".join(item) for item in grid])
            return

        max_col = len(grid[row])
        for col in range(max_col):
            if not self.is_valid(grid, row, col):
                continue

            grid[row][col] = "Q"
            self.bachtrace(grid, row + 1)
            grid[row][col] = "."

    def solveNQueens(self, n: int) -> List[List[str]]:
        self.result = list()
        grid = [["." for _ in range(n)] for _ in range(n)]
        self.bachtrace(grid, 0)
        return self.result
