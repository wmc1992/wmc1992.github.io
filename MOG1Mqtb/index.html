209. 长度最小的子数组#

链接: https://leetcode.cn/problems/minimum-size-subarray-sum/description/

1、题目描述#

给定一个含有 n 个正整数的数组和一个正整数 target 。

找出该数组中满足其总和大于等于 target 的长度最小的 连续子数组 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。

示例 1：

输入：target = 7, nums = [2,3,1,2,4,3]

输出：2

解释：子数组 [4,3] 是该条件下的长度最小的子数组。

示例 2：

输入：target = 4, nums = [1,4,4]

输出：1

示例 3：

输入：target = 11, nums = [1,1,1,1,1,1,1,1]

输出：0

提示：

  • 1 <= target <= 109
  • 1 <= nums.length <= 105
  • 1 <= nums[i] <= 105

进阶：

如果你已经实现 O(n) 时间复杂度的解法, 请尝试设计一个 O(n log(n)) 时间复杂度的解法。

2、思路说明#

该题目使用滑动窗口解法。

右侧指针每次向右移动一个位置。主要是左侧指针什么时候向右移动，左侧指针向右移动的时机为：当窗口内的所有元素之和大于等于 target 时。

另外需要注意的是：每次向右移动左指针时，都要判断当前窗口内所有元素之和是否大于target，如果大于就要更新窗口的长度。

3、代码#

from typing import List

class Solution:

    def is_valid(self, target, arr):
        return sum(arr) >= target

    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        start, end = 0, -1

        min_len = float("inf")
        while start < len(nums) and end < len(nums):
            end += 1
            valid_flag = self.is_valid(target, nums[start:end+1])

            # 如果当前窗口的值没有大于等于target
            if not valid_flag:
                continue

            # 如果当前窗口的值已经大于等于了target
            if end - start + 1 < min_len:
                min_len = end - start + 1
            while valid_flag and start < end:
                start += 1
                valid_flag = self.is_valid(target, nums[start:end+1])
                if valid_flag and end - start + 1 < min_len:
                    min_len = end - start + 1

        if min_len == float("inf"):
            return 0
        return min_len
