647. 回文子串#

链接: https://leetcode.cn/problems/palindromic-substrings/

1、题目描述#

给你一个字符串 s ，请你统计并返回这个字符串中 回文子串 的数目。

回文字符串 是正着读和倒过来读一样的字符串。

子字符串 是字符串中的由连续字符组成的一个序列。

具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被视作不同的子串。

示例 1：

输入：s = "abc"

输出：3

解释：三个回文子串: "a", "b", "c"

示例 2：

输入：s = "aaa"

输出：6

解释：6个回文子串: "a", "a", "a", "aa", "aa", "aaa"

提示：

  • 1 <= s.length <= 1000
  • s 由小写英文字母组成

2、思路说明#

使用动态规划解决该问题。dp 数组含义为：dp[i][j] 表示 s[i:j] 是否为回文串。这样，该问题就变为了只要遍历填充完整个 dp 数组之后，统计一下 dp 数组中的为回文串的数量就可以了。

3、代码#

def process(s):
    n = len(s)

    # dp数据的含义：dp[i][j] 是否回文子串
    # 对于该题目，只要将这个 dp 全部填充一遍，然后统计其中为1的数量即可
    dp = [[0 for _ in range(n)] for _ in range(n)]

    # 对于回文子串，迭代过程为 s[i:j] <- s[i+1:j-1]
    # 所以 i 是从大到小遍历，j 是从小到大遍历
    for i in range(n - 1, -1, -1):
        for j in range(i, n, 1):

            # 单个字符肯定是回文串            
            if i == j:
                dp[i][j] = 1

            # 相邻两个字符也是回文串
            elif j - i == 1:
                dp[i][j] = 1

            # 超过两个字符之后就按照递归条件判断
            else:
                if s[i] != s[j]:
                    dp[i][j] = 0
                else:
                    if dp[i+1][j-1] == 1:
                        dp[i][j] = 1
                    else:
                        dp[i][j] = 0

    counter = 0
    for i in range(0, n, 1):
        for j in range(i, n, 1):
            if dp[i][j] == 1:
                counter += 1
    return counter
