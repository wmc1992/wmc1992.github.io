746. 使用最小花费爬楼梯#

链接: https://leetcode.cn/problems/min-cost-climbing-stairs/

1、题目描述#

给你一个整数数组 cost ，其中 cost[i] 是从楼梯第 i 个台阶向上爬需要支付的费用。一旦你支付此费用，即可选择向上爬一个或者两个台阶。

你可以选择从下标为 0 或下标为 1 的台阶开始爬楼梯。

请你计算并返回达到楼梯顶部的最低花费。

示例 1：

输入：cost = [10,15,20]

输出：15

解释：你将从下标为 1 的台阶开始。

  • 支付 15 ，向上爬两个台阶，到达楼梯顶部。

总花费为 15 。

示例 2：

输入：cost = [1,100,1,1,1,100,1,1,100,1]

输出：6

解释：你将从下标为 0 的台阶开始。

  • 支付 1 ，向上爬两个台阶，到达下标为 2 的台阶。

  • 支付 1 ，向上爬两个台阶，到达下标为 4 的台阶。

  • 支付 1 ，向上爬两个台阶，到达下标为 6 的台阶。

  • 支付 1 ，向上爬一个台阶，到达下标为 7 的台阶。

  • 支付 1 ，向上爬两个台阶，到达下标为 9 的台阶。

  • 支付 1 ，向上爬一个台阶，到达楼梯顶部。

总花费为 6 。

提示：

  • 2 <= cost.length <= 1000
  • 0 <= cost[i] <= 999

2、思路说明#

该题目需要理解清楚题意，cost 中保存的数值是离开该台阶时需要的花费，所以进入0台阶和1台阶是没有花费的。另外，题目中的到达顶部指的是数组 cost 之外的位置。

dp 数据的含义：dp[i] 表示达到台阶 i 处的花费。

3、代码#

from typing import List

class Solution:

    def minCostClimbingStairs(self, cost: List[int]) -> int:
        # dp 数组的含义为到达台阶 i 处所需要的花费
        # dp 数组的长度为 n + 1，题目中的到达顶部指的是到达索引为 n 处的台阶
        dp = [0 for _ in range(len(cost) + 1)]

        dp[0] = 0
        dp[1] = 0

        for i in range(2, len(cost) + 1, 1):
            c1 = dp[i - 1] + cost[i - 1]
            c2 = dp[i - 2] + cost[i - 2]

            dp[i] = c1 if c1 < c2 else c2

        return dp[len(cost)]
