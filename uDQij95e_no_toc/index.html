111. 二叉树的最小深度#

链接: https://leetcode.cn/problems/minimum-depth-of-binary-tree/

1、题目描述#

给定一个二叉树，找出其最小深度。

最小深度是从根节点到最近叶子节点的最短路径上的节点数量。

说明：叶子节点是指没有子节点的节点。

示例 1：

输入：root = [3,9,20,null,null,15,7]

输出：2

示例 2：

输入：root = [2,null,3,null,4,null,5,null,6]

输出：5

提示：

  • 树中节点数的范围在 [0, 105] 内
  • -1000 <= Node.val <= 1000

2、思路说明#

这个题目和二叉树的最大深度思路是相同的，也是使用后序遍历，这样在处理根节点时，左右子树的最小深度都是已知的了，取其中的较小值然后加1就是当前节点的最小深度。

和二叉树的最大深度那个题目类似，需要注意不要使用下述这种处理边界条件的方式：

def recursive(node):
    if node is None:
        return 0
    ...

在该题目中，叶子节点才是边界条件，叶子节点的深度为1。如果采用上述代码作为边界条件，那么如果某一个节点没有左孩子，只有右孩子，那么按照上述代码就会出现：当前节点的右孩子深度为真实值，而当前节点的左孩子深度为0。这里的 "左孩子深度为0" 是错误的。

3、代码#

from typing import Optional

# Definition for a binary tree node.
class TreeNode:

    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:

    def minDepth(self, root: Optional[TreeNode]) -> int:
        # 边界条件
        if root is None:
            return 0

        # 对于叶子节点，其深度为1
        if root is not None and root.left is None and root.right is None:
            return 1

        if root.left is not None and root.right is not None:
            left_deep = self.minDepth(root.left)        
            right_deep = self.minDepth(root.right)
            deep = left_deep if left_deep < right_deep else right_deep
            return deep + 1
        elif root.left is not None:
            return self.minDepth(root.left) + 1
        else:
            return self.minDepth(root.right) + 1
